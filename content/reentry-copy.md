# Re-entry copy (мягкое возвращение)

Используй эти тексты, когда пользователь пропустил день/несколько дней.
Тон: поддержка, никакого стыда, никакой «дисциплинарной» риторики.

## Базовые фразы

- Пауза — это не провал. Возвращаемся с одного микрошагa.
- Ничего не сломано. Твой ритм можно перезапустить мягко.
- Даже 60 секунд сегодня уже меняют траекторию.
- Не нужно догонять «пропущенное». Достаточно одного шага сейчас.
- Ты снова здесь — этого уже достаточно для нового старта.

## Если пропущен 1 день

- Вчера был паузой. Сегодня — лёгкий рестарт.
- Возвращение после одного пропуска — идеальный момент для микрошагa.
- Без компенсации и без наказаний: просто один шаг.

## Если пропущено 2+ дня

- После паузы начинаем с минимума, не с максимума.
- Чем меньше сил, тем важнее короткий и выполнимый шаг.
- Мягкий вход лучше резкого рывка.

## CTA для re-entry

- ⚡ Вернуться мягко (1 шаг)
- Сделать короткий рестарт
- Начать заново без давления

## Чего НЕ писать

- «Ты сорвал серию»
- «Нужно наверстать»
- «Опять пропустил(а)»
- Любые формулировки с виной/стыдом
