# PROJECT ENERGY

Одностраничный лендинг-концепт в стиле **Calm Minimal** для программы «Перезапуск энергии за 7 дней».

## Что внутри
- Заголовок: **PROJECT ENERGY**
- Подзаголовок: «Перезапуск энергии за 7 дней»
- 3 смысловые карточки: Питание, Среда, Восстановление
- CTA-кнопка: «Начать 7-дневный цикл»

## Стек
- HTML5
- CSS3 (адаптивная верстка, без сборки)

## Локальный запуск
Открой `index.html` в браузере двойным кликом
или подними простой сервер:

```bash
python3 -m http.server 8080
```

Далее открой: `http://localhost:8080`
