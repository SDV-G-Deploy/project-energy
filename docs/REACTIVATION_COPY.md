# Reactivation copy — D1 / D3 / D7

Шаблоны для in-app блока **«Вернуться в ритм»**.

> Канал: только внутри продукта (без push, без email/telegram отправок).

---

## D1 (нет completion 1 день)

- Заголовок: `D1 · Пауза 1 день`
- Текст:
  - `Один день паузы — это нормально. Вернуться можно с одного очень лёгкого шага.`
- Primary CTA:
  - `Сделать 1 мягкий шаг сейчас`
- Secondary CTA:
  - `Вернусь позже сегодня`

## D3 (нет completion 3 дня)

- Заголовок: `D3 · Пауза 3 дня`
- Текст:
  - `Ритм просел, но не сломан. 60 секунд мягкого действия помогут вернуть инерцию.`
- Primary CTA:
  - `Сделать 1 мягкий шаг сейчас`
- Secondary CTA:
  - `Вернусь позже сегодня`

## D7 (нет completion 7+ дней)

- Заголовок: `D7 · Пауза 7+ дней`
- Текст:
  - `Даже после длинной паузы всё можно перезапустить без давления: один микрошаг сейчас.`
- Primary CTA:
  - `Вернуться мягко за 60 сек`
- Secondary CTA:
  - `Вернусь позже сегодня`

---

## Персонализированная рекомендация (в блоке)

Показывается под основным текстом:

- на основе последнего completion (`actionId` / последний сработавший шаг),
- с учётом последнего якоря intent (если пользователь выбирал «После подъёма», «Перед сном» и т.п.).

Пример:

- `Рекомендация: Начни с того, что уже срабатывало: «Дыхание 4/6 (1 минута)». Якорь из намерений: после подъёма.`

---

## Локальные события аналитики

- `reactivation_prompt_shown`
  - stage (`D1|D3|D7`)
  - daysWithoutCompletion
- `reactivation_prompt_action`
  - action (`start_now` / `snooze_today`)
  - stage
